  • Specfun.bernoa - Compute Bernoulli number Bn, using recurrence relation.

  • Specfun.eulera - Compute Euler number En, using recurrence relation.

3 Gamma, Beta, and Psi Functions

  • Specfun.gamma2 - Compute gamma function Г(x).
  • Specfun.lgama - Compute gamma function ln[Γ(x)] or Γ(x).
  • Specfun.cgama - Compute complex gamma function ln[Г(z)] or Г(z).
  • Specfun.beta - Compute beta function B(p, q).
  • Specfun.psi - Compute Psi function Ψ(x) (Digamma Function).
  • Specfun.cpsi - Compute complex psi function Ψ(z) (Digamma Function).
  • Specfun.incog - Compute incomplete gamma function γ(a,x), Γ(a,x) and P(a,x).
  • Specfun.incob - Compute incomplete beta function Ix(a,b).
